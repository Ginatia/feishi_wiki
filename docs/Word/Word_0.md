# 创建自动图文集

## 创建复用模板

1. 找到文档里那个格式最标准的内容，如表题注
2. 精准选中它：选中整行，但注意不要选中后面的换行符
3. 按下快捷键 Alt + F3
4. 在弹出的 新建构建基块 窗口中设置 名称，库（自动图文集），保存位置（默认 Normal.dotm）
5. 点击确定

## 给这个模板绑定快捷键

1. 点击 文件 -> 选项 -> 自定义功能区
2. 点击底部的 键盘快捷键：自定义
3. 在左侧“类别”框里，拉到最下面找到 “构建基块”
4. 在右侧 “构建基块” 框里，找到刚才起的名称
5. 点击 请按新快捷键下方的输入框，并按下组合快捷键
6. 点击左下角指定
7. 在需要使用模板时，按下快捷键即可
